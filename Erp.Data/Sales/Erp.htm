销售单（SaleOrder）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
CustomerId 客户 Int32 N
Title 标题 String 200 概要描述信息
Quantity 数量 Int32 N 总件数
Price 价值 Decimal N 产品总价加上运费
Status 状态 Int32 N
OccurTime 发生时间 DateTime
ContractNo 合同编号 String 50
Operator 经办人 String 50
Invoice 发票 Boolean N 已开票
BillCode 快递单号 String 50 发货的快递单号，多个逗号隔开
Freight 运费 Decimal N 快递费
Receiver 收件人 String 50
Phone 收件电话 String 50
Address 收件地址 String 50
CreateUser 创建者 String 50
CreateUserID 创建人 Int32 N
CreateIP 创建地址 String 50
CreateTime 创建时间 DateTime
UpdateUser 更新者 String 50
UpdateUserID 更新人 Int32 N
UpdateIP 更新地址 String 50
UpdateTime 更新时间 DateTime
Remark 备注 String 500
